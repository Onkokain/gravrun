# Gravrun

Gravrun is a first person 3D parkour game where movement revolves around manipulating gravity. Instead of being limited to one direction you, you can instantly redirect gravity toward the surface you're looking at, allowing you to run across walls, ceilings, and other structures. Every level is designed around this mechanic, requiring precise movement, timing, and skills to move towards the exit.

The game focuses on momentum-based movement rather than combat. As the levels become more complex, you'll need to combine gravity shifts, and jumps, avoid hazards, and discover the fastest route through each environment.

## Gameplay

The core mechanic of Gravrun is gravity switching. By looking at a nearby surface and pressing **Q**, gravity immediately rotates toward that surface, allowing you to continue moving as if it were the new floor. This simple mechanic opens up a wide variety of movement possibilities, from wall running and ceiling traversal to chaining multiple gravity flips together in quick succession.

Each of the ten levels introduces new challenges that test your understanding of the movement system. Success depends on mastering momentum, planning your route, and reacting quickly to changing environments.

## Features

* First-person movement with smooth controls
* Gravity switching onto walls, ceilings, and floors
* Momentum-based parkour gameplay
* Ten handcrafted levels with increasing difficulty
* Fast-paced traversal focused entirely on movement
* Built using the Godot Engine with Jolt Physics

## Controls

| Key               | Action                                              |
| ----------------- | --------------------------------------------------- |
| WASD / Arrow Keys | Move                                                |
| Mouse             | Look around                                         |
| Space             | Jump                                                |
| Q                 | Switch gravity toward the surface you're looking at |
| Shift             | Toggle sprint                                       |
| Esc               | Pause Menu                                          |

## Running Locally

### Requirements

* Godot 4.7 or newer

### Setup

1. Clone or download this repository.
2. Open the project by selecting `project.godot` in Godot.
3. Press **F5** or click **Run Project** to start the game.

## Project Structure

The project is developed entirely in Godot using GDScript. Physics simulation is handled by the Jolt Physics extension, while gameplay logic, player movement, gravity manipulation, and level systems are implemented within the project.
